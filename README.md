# AI Website Builder

An AI-powered web application that generates complete websites from text prompts.

## Features
- Generate websites from natural language prompts  
- AI-generated HTML, CSS, and JavaScript  
- Automatic image generation for website design  
- Live website preview in browser  
- Open generated website in a separate tab  

## Tech Stack
- Python  
- FastAPI  
- Hugging Face API  
- HTML, CSS, JavaScript  
- Stable Diffusion XL  

## How It Works
1. User enters a website prompt  
2. AI generates website code  
3. AI creates matching images  
4. Final website is rendered instantly  

## Installation

```bash
pip install fastapi uvicorn python-dotenv huggingface_hub
```
